Applied Geospatial Techniques (AGT)

Applied Geospatial Techniques (AGT) is a hands on practical subject done in both geospatial science and surveying. The subject uses the knowledge from these three different disciplines to create a series of products:
  • Surveying
  • Geographical Information Systems (GIS)
  • Cartography

For the 2021 version of this course, the assignment was about the proposal of creating an amphitheatre at Deep Rock Oval located at Yarra Bend Park. The main deliverables of the subject consisted of producing:

  • Topographic survey map of Deep Rock Oval
  • 3D generated scene of the proposed amphitheatre
  • Amphitheatre and Pathway plan of the proposed site
  • Final map of the amphitheatre proposal

The topographic survey map would have been created using surveying data obtained on the field using a total station at Deep Rock Oval, Yarra Bend Park. However, because of COVID19 only a visit to the site was viable, with the data being provided to us and the surveying experience transfered to another subject. The topographic map was made using LISCAD.SEE and LISCAD.CAD. The 3D generated scene was created using the ArcGIS suite while the raw data was modified in ArcMap. The proposed pathway was created using ArcMap by using the GIS program to generate the pathway based on slope angle and the amphitheatre based on z-values. The final map used Adobe Illustrator to create the map along with ArcMap for exporting the 2D amphitheathre image and for exporting the image of the 3D scene.

Spatial Infomation Science Analytics

Spatial Information Science Analytics is a difficult, content ritch, subject that was difficult to master. Undertaking this course was a trial of many late nights, and hours dedicated to understanding complex spatial methods. There were many practical aspects of this course, using ARCMAP to make different map products utilising different general geospatial topics including:
  • Spatial Interpolation
  • Spatial Pattern Analysis
  • Spatial Regression Analysis

In 2021, this course had a major focus on the completion of a full mapping project from start to finish and included a full project proposal and various other planning documents designed to support the final product of this project. For this project I decided to use my newfound knowledge of Spatial interpolation to generate a map of fire vulerable areas in the state of victoria using weather station data as a basis of these maps. This was further augmented through the use of other data sources such as a map of forested area and various other metrics.
